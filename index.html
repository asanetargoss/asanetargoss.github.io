asanetargoss' Mostly Minecraft

Hi, I'm asanetargoss. I play Minecraft (modded, Java) and do mostly Minecraft-related things under this alias. I've decided I need a more permanent internet presence where I can write pretty much whatever. Coremodding and Minetest immediately come to mind, although there will probably be Minecraft, coding, and creating stuff on a computer in general. Or maybe hardly anything at all. I haven't really planned out this whole website thing.

Here's where I exist:

  • GitHub - Where I send pull requests to open-source mods, and slowly hack away at a mod of my own for a hardcore/survival/magic modpack.
  • Minecraft Forum - Mostly fanfiction and the occasional forum post. I'm currently most active in a crossover Minecraft fanfiction known as The Convergence. The three of us currently working on it are inarguably the most active members of the Literature Section. It's pretty quiet.
  • Twitter - Where I occasionally announce things related to my modpack and sometimes have a personality.

Here are other things that exist:

  • Feed the Beast subreddit - Notable for being the hub of the modded Minecraft community.
  • Git - If you are writing Minecraft mods, use Git, ideally using branches forked from your existing Git history. And release dev jars. Please. ♡
  • MultiMC - The best modded Minecraft launcher and arguably the best vanilla launcher.
  • Minetest - Always on my radar. It's the most viable Minecraft-like with potential for modding.
  • culegooner's Coremodding Tutorial - Coremodding in a nutshell aside from @IFMLLoadingPlugin.SortingIndex(value = 1001). I might do a write-up about coremodding at some point.
  • Bytecode Outline - Helpful Eclipse plugin for understanding Java bytecode and ObjectWeb ASM.
  • Java Decompiler - The IDE plugin makes working with random mods and source-less jars a lot easier.
  • CodeChicken Core for 1.8/1.9/1.10 - Another helpful tool which actually makes using non-dev mod jars in a dev environment possible.
