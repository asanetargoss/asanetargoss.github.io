Hi, I'm asanetargoss. I play Minecraft (modded, Java) and do mostly Minecraft-related things under this alias. Here you'll find some things I do under this alias, and links I find useful.

Below is some of the stuff I've worked on:

  • The Convergence - Me and a few other Minecraft fanfiction writers get together to write this crossover story in which characters from our own story worlds join together to fight a mysterious evil.

I am also on Twitter, which I post on occasionally. You may also be interested in my GitHub.

Below is a list of Minecraft-related things I find useful, mostly related to modding:

  • Feed the Beast subreddit - Notable for being the hub of the modded Minecraft community.
  • Git - If you are writing Minecraft mods, use Git, and try to stick to one change per commit.
  • MultiMC - The best modded Minecraft launcher and arguably the best vanilla launcher.
  • Minetest - Always on my radar. It's the most viable Minecraft-like with potential for modding.
  • culegooner's Coremodding Tutorial - Coremodding in a nutshell aside from @IFMLLoadingPlugin.SortingIndex(value = 1001). I might do a write-up about coremodding at some point.
  • Mod Coder Pack (MCP) - Contains the MCP Mapping Viewer, a handy tool that lets you look up what class/field/method names are depending on the environment.
  • Bytecode Outline - Helpful Eclipse plugin for understanding Java bytecode and ObjectWeb ASM. Fair warning: the variable stack prediction feature was buggy the last time I used it.
  • Java Decompiler - The IDE plugin makes working with random mods and source-less jars a lot easier.
  • CodeChicken Core for 1.8/1.9/1.10 - Another helpful tool which actually makes using non-dev mod jars in a dev environment possible. Also available for other versions.
